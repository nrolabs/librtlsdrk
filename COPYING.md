# librtlsdrk

Kotlin port of [librtlsdr (RTL-SDR Blog fork)](https://github.com/rtlsdrblog/rtl-sdr-blog)
for the Android USB host API.

Canonical repository: <https://github.com/nrolabs/librtlsdrk>
